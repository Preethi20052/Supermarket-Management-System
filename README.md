# 🛒 Supermarket Management System

💻 Full Stack Web Application  
🚀 Spring Boot | React | MySQL | Java  

---

## 📌 About The Project

The Supermarket Management System is a full-stack web application inspired by modern e-commerce platforms like Amazon and quick-commerce services like Zepto.

It provides a seamless shopping experience where users can browse products, add items to cart, and place orders efficiently. The system also allows admins to manage inventory, track orders, and handle billing operations.

A unique feature of this project is Society Bulk Ordering, which enables multiple users from the same society or community to place combined orders, reducing delivery costs , provide discount on products and improving efficiency.

---

## ⚙️ Key Features

- Secure User Authentication  
- Product Management (Add / Update / Delete)  
- Real-time Inventory Tracking  
- Cart and Order Management  
- Billing and Checkout System  
- Invoice Generation  
- Fast and scalable backend using Spring Boot  

### 🌟 Unique Feature

Society Bulk Ordering:
- Users from the same society can join a common order  
- Orders are grouped together for bulk purchasing  
- Reduces delivery charges  and provide discount on products
- Improves delivery efficiency

## 🚀 How It Works

1. User registers and logs into the system  
2. Admin manages product inventory  
3. Users browse products and add items to cart  
4. Users can place individual orders or join a society bulk order  
5. System calculates total bill dynamically  
6. Order is processed .

## ▶️ Installation & Setup

Clone the repository:
git clone https://github.com/Preethi20052/Supermarket-Management-System.git
cd Supermarket-Management-System

Backend Setup (Spring Boot):
cd backend
./mvnw spring-boot:run

Frontend Setup (React):
cd frontend
npm install
npm start

Database Setup (MySQL):
- Create a database in MySQL  
- Update application.properties with your credentials  
- Run SQL scripts if available
